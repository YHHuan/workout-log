# 💪 訓練日誌 - 個人健身紀錄應用

一個簡潔、功能完整的訓練日誌網頁應用，幫助您記錄每天的重訓進度。

## 功能特性

✅ **18 個預設訓練動作**
- 5 個主要訓練動作（腿部推、垂直推、腿部拉、水平拉、水平推）
- 5 個輔助核心訓練
- 8 個跑者膝蓋復健動作

✅ **靈活的紀錄系統**
- 簡潔的「重量、組數、次數」輸入框
- 摺疊式紀錄區塊，保持版面清爽
- 自動保存紀錄到瀏覽器本地存儲

✅ **單位轉換**
- 支持公斤 (kg) 與磅 (lb) 自由切換
- 紀錄自動轉換，無需手動計算

✅ **自定義動作**
- 隨時新增自己的訓練動作
- 為每個動作添加名稱、分類和要領

✅ **歷史紀錄追蹤**
- 每個動作都保留完整的訓練歷史
- 方便查看進度和調整訓練計畫

## 使用方法

### 在本地使用

1. 下載或複製所有文件到本地資料夾
2. 用瀏覽器打開 `index.html` 文件即可使用

### 部署到 GitHub Pages（推薦永久方案）

1. **建立 GitHub 帳號**（如果還沒有的話）
   - 訪問 [github.com](https://github.com) 並註冊

2. **建立新的 Repository**
   - 點擊 "New" 建立新倉庫
   - 命名為 `workout-log`（或任何您喜歡的名稱）
   - 選擇 "Public" 以啟用 GitHub Pages
   - 點擊 "Create repository"

3. **上傳文件**
   - 在新建的倉庫中，點擊 "Add file" → "Upload files"
   - 上傳 `index.html`、`style.css` 和 `script.js` 三個文件
   - 點擊 "Commit changes"

4. **啟用 GitHub Pages**
   - 進入倉庫的 "Settings" 標籤
   - 左側選擇 "Pages"
   - 在 "Source" 下選擇 "main" 分支
   - 點擊 "Save"
   - 稍等幾分鐘後，您會看到一個公開的 URL（例如：`https://yourusername.github.io/workout-log`）

5. **完成！**
   - 現在您可以在任何設備上訪問這個永久連結
   - 您的訓練紀錄會安全地保存在瀏覽器中

## 數據存儲

所有訓練紀錄都保存在您的瀏覽器的 **LocalStorage** 中，這意味著：
- ✅ 您的數據完全私密，不會上傳到任何服務器
- ✅ 即使離線也能使用
- ✅ 清除瀏覽器數據時會被刪除（建議定期備份）

## 瀏覽器相容性

支持所有現代瀏覽器：
- Chrome / Edge
- Firefox
- Safari
- 手機瀏覽器（iOS Safari、Chrome Mobile 等）

## 常見問題

**Q: 我的紀錄會丟失嗎？**
A: 只要您不清除瀏覽器的 LocalStorage，紀錄會一直保存。建議定期備份。

**Q: 能在多個設備上同步嗎？**
A: 目前不支持自動同步。如需跨設備使用，建議在每個設備上分別打開網頁。

**Q: 可以修改預設動作嗎？**
A: 可以！您可以刪除任何預設動作，或新增自己的動作。
